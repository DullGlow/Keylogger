# Keylogger
Simple Keylogger developed in 32-bit NASM. Tested on Ubuntu 20.04 64-bit
## Requirements
You will need Linux with NASM assembler

## Installation:

    git clone https://github.com/DullGlow/Keylogger.git
    cd Keylogger
    make
    sudo ./keylogger
